# Loja
Esse projeto, tem o intuito de: Manter cadastro de clientes;
Manter cadastro de fornecedores;
Manter cadastro de funcionários;
Cadastrar produtos;
Busca dos produtos disponibilizados pelas lojas selecionadas como favoritas;
Emitir relatório de vendas;
Notificação sobre novos produtos(Não concluido);
Sistema de avaliação de atendimento e entrega;
Notificação sobre a entrega de produtos(Não concluido);
Manter vendas;

Linguagem de Programação: Java. 
Recursos utilizados: MVC POO. 
Banco: MYSQL.
